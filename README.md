## 📌 Project Overview

LapExpress is a full-stack **Laptop E-Commerce web application** built using Django. It allows users to browse laptops, add products to cart, and interact with a structured authentication and product management system.

The project demonstrates real-world features like **authentication, role-based access control, cart system, and admin management**, making it suitable for production-level learning.

---

## 🚀 Core Features

### 👤 User Features

* User registration & login system
* Password reset via email authentication
* Browse laptop products with details
* Add to cart functionality
* Responsive and user-friendly UI

👉 **In Progress:**

* Buy Now & billing system
* Order/checkout functionality

---

### 🛠️ Admin / Staff Features

* Secure Django admin panel (superuser access)
* Role-based access control:

  * Superuser can assign staff roles
  * Staff users have limited permissions
* Add, update, and delete laptop products
* Manage product listings efficiently

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```bash id="1x0x8m"
git clone https://github.com/your-username/lapexpress-ecommerce.git
cd lapexpress-ecommerce
```

### 2️⃣ Create Virtual Environment

```bash id="y70ilp"
python -m venv venv
venv\Scripts\activate
```

### 3️⃣ Install Dependencies

```bash id="x4r6m0"
pip install -r requirements.txt
```

### 4️⃣ Run Server

```bash id="c0r4d7"
python manage.py runserver
```
